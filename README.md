# Sims4 Yootz Story

*(aka my save in the sims 4 where everything went crazy)* 

This started off as me simply exploring the game back when it was relatively new, with 2 adult sims that were roommates, but became my main save file that I found myself returning to over and over. In typical sims fashion, everything got weird and wacky. 

**Please note:** this is an early rough first draft, everything within is subject to change. I'm still working things out, even major plot points & worldbuilding. 

## How To Read This Story

Above this part, click on "Chapters" folder, and then click on the md file whatever chapter ("##-ChapterName.md") you'd like to read. Github will open the markdown file as a rendered preview (so with all the images & text together) for you. Or you can download the whole project & open it in your favorite markdown software (mine is obsidian). Your choice.
